## Inspired by [changkun's midgard project](https://github.com/changkun/midgard)

## Features

1. data backup
2. data share between more one devices
3. multiusers

## How To Run

```shell
1. git clone https://github.com/dmzlingyin/clipshare.git
2. cd clipshare
3. make
4. ./clipshare
```
